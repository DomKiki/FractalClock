Projects > Maths > Fractal Clock

Theory

Fractals are figures exhibiting similar patterns at increasingly small scales (called self similarity).
Building one is really simple :

  • Draw a line of length L
  • Rotate by some angle θ
  • Shorten L by some factor f (L = L * f)
  • Repeat until L is less than some threshold

We will end up with a spiral-shaped figure. Now if instead of drawing one branch at each intersection we draw more (with different θ) we will get a tree-like figure.

For this one, θ is the three clock's hands angle (relative to the origin).

Controls

  • Shrink : Shortening factor f
  • Min. Length : Minimum length of the branches
  • Branches :
    1. Seconds
    2. Seconds / Minutes
    3. Seconds / Minutes / Hours
  • Color : Individual (each angle has its color) / None (all black)
  • Hide/Show Clock : En/disable clock's display
